# Волна — личный музыкальный плеер 🎵

Приложение для прослушивания музыки без рекламы, с перемоткой и работой в фоне.
Работает как PWA: устанавливается на iPhone прямо из Safari.

**Источники музыки:**
- **Audius** — официальное открытое API, легально и стабильно.
- **YouTube** — через публичные серверы Piped/Invidious (почти вся русская музыка).

---

## Как запустить (GitHub Pages, ~5 минут)

1. Зайди на [github.com](https://github.com) и войди в свой аккаунт.
2. Нажми **+** (вверху справа) → **New repository**.
   - Repository name: `volna` (или любое другое).
   - Оставь **Public**.
   - Нажми **Create repository**.
3. На странице нового репозитория нажми ссылку **uploading an existing file**.
4. Перетащи туда **все файлы из этой папки**:
   `index.html`, `app.js`, `sw.js`, `manifest.webmanifest`, `icon-180.png`, `icon-512.png`, `README.md`
5. Нажми **Commit changes**.
6. Открой **Settings** (вкладка репозитория) → слева **Pages**.
   - Source: **Deploy from a branch**.
   - Branch: **main**, папка **/ (root)** → **Save**.
7. Подожди 1–2 минуты. Адрес приложения появится там же, вида:
   `https://ТВОЙ_НИК.github.io/volna/`

## Установка на iPhone

1. Открой этот адрес в **Safari** (важно: именно Safari).
2. Нажми кнопку **Поделиться** (квадрат со стрелкой).
3. Выбери **«На экран “Домой”»** → **Добавить**.
4. Готово — «Волна» появится на рабочем столе как обычное приложение.

Музыка играет в фоне и при заблокированном экране, управление — с экрана
блокировки (play/pause, следующий трек, перемотка).

## Если YouTube перестал играть

Публичные серверы Piped/Invidious периодически падают или блокируются YouTube.
Это нормально, лечится так:

1. Открой в приложении вкладку **Настройки** → нажми **Проверить** — увидишь,
   какие серверы живы.
2. Если все мертвы — найди свежие адреса в официальном списке инстансов Piped
   (github.com/TeamPiped/documentation → public instances) или Invidious
   (docs.invidious.io/instances) и добавь их в настройках (по одному на строку) → **Сохранить**.
3. Audius при этом продолжает работать всегда — он не зависит от этих серверов.

## Важно знать

- Все данные (избранное, настройки) хранятся только на твоём телефоне.
- Аккаунты, реклама, трекинг — отсутствуют.
- Воспроизведение с YouTube через сторонние серверы — «серая зона» с точки
  зрения правил YouTube. Для личного использования риск минимален, но
  распространять приложение публично как сервис не стоит.

## Обновление приложения

Изменил файлы в репозитории → GitHub Pages сам обновит сайт через минуту-две.
На телефоне достаточно закрыть и заново открыть приложение.
